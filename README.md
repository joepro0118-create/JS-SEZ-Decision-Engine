# Nexus JS-SEZ Strategic Architect 🌐✨

Nexus JS-SEZ Strategic Architect is a conceptual full-stack web application developed for UM Hackathon 2026. The platform aims to serve as an intelligent decision-support and strategic planning tool.

## Website Link 🔴
[https://js-sez-decision-engine.vercel.app](https://js-sez-decision-engine.vercel.app)

## Tech Stack 💻

### Frontend
- **React 19**
- **Vite 8**
- **Tailwind CSS 4**
- **Framer Motion** for animations
- **Lucide React & React Icons** for UI iconography
- **React Router DOM** for navigation
- **React Markdown** for rich text rendering

### Backend
- **Node.js** with **Express**
- **@google/generative-ai** (Gemini)
- **OpenAI API**
- **Axios**
- **dotenv** & **cors**

## Getting Started 🚀

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd JS-sez-twinning
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   ```
   Create a `.env` file in the `backend` directory based on required environment variables (e.g., `GEMINI_API_KEY`, etc.).
   
   Start the backend server:
   ```bash
   npm run dev
   ```

3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install
   ```
   Start the frontend development server:
   ```bash
   npm run dev
   ```
